# guia-algoritmo-notebookLM
Projeto desenvolvido como desafio da DIO que utiliza NotebookLM para criar um assistente de estudos. Meu objetivo foi criar um caderno para aprender algoritmos e lógica de programação com apoio de IA.

  Prompts utilizados:
- Explique o que é um algoritmo para um iniciante.
- Dois exemplos de algortimos do dia a dia.
- Por que a lógica é importante na programação?

  Dificuldade encontrada:
Uma das dificuldades foi que algumas respostas geradas eram muito extensas. Para melhorar a experiência de estudo, foi necessário ajustas os prompts para obter respostas mais curtas, diretas e didáticas sem perder a clareza das explicações.

  Resumo:
  Um algoritmo é uma sequência de isntruções para resolver um problema ou realizar uma tarefa. Algoritmos estão presentes em diversas partes do nosso dia a dia, desde a receita de um bolo até os vídeos que assistimos nas redes sociais.
Para um programador eles são fundamentais, pois antes mesmo de começar a escrever um código é necessário estruturar a sequência de passos que será seguida para resolver um problema. 
  Prompts reutilizáveis:
- Explique de forma simples e direta para um iniciante em programação.
- Quais são os exemplos práticos no dia a dia?
- Por onde começar sendo um inicante.
